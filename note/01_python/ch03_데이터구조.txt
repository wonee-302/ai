3장 데이터구조: 여러데이타를 한 변수에 담는게 목적 
1. 리스트: [타입에 상관없이 0개 이상의 데이터들] ['서울', '대전', '부산']
	1.1 리스트 생성: []. list(나열 가능한 자료) list((from, to, step) )
	1.2 기본정보조회
 		- lst[idx]
		- min(lst), max(lst), sum(lst), len(lst)
    		- sorted(lst, reverse=True): 정렬된 lst를 반환 
		- lst.sort(reverse=True):lst가 정렬된 값으로 변경 
	1.3 요소추가 
		lst.append(value), extend([values], insert(idx, value)
	1.4 인덱싱과 인덱싱 찾기 
		-lst[idx]
		-index(찾을 데이터): 없는 데이터를 찾을 경우 error 
		-찾을 데이터 in lst : 찾을 데이터가 있는지 여부 반환 						
		-lst.count(찾을 데이터): 찾울 데이터가 몇개인지 숫자 반환 
	1.5 원하는 요소 추출 	
		-슬라이싱	
		-리스트 컴프리핸션(원하는 데이터 추출) 
	1.6 요소수정 : 인덱싱이나 슬라이싱을 이용 
	1.7 요소삭제:pop(), pop(idx), del lst[idx], remove(데이터) 
	1.8 복제(깊은 복사: 복제할변수 = 원본데이터.copy()
