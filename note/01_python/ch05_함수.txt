5장 함수 
1절. 함수의 정의 및 사용(def 함수명(매개변수..) :~)
	-오버로딩(같은 이름의 함수를 매개변수를 달리 사용) 불가 
	-docstring(함수 맨 윗줄 설명을 덧붙여 사용 용이)
	-전역변수 vs 지역변수(global을 이용하여 함수네에서 전역변수 사용)
2절. 함수의 결과를 반환하는 return
	-return a, b - (a,b) 튜플을 return 
3절. 함수의 매개변수 
	-매개변수에 기본값을 가질 때 : 함수 호출시 선택적으로 명시. 정의 할때 위치가 뒤로 가야한다. 
		*기본값이 스칼라변수: 기본값은 함수정의 시점 
		*기본값이 데이터 구조, 객체: 기본값은 함수호출시점 (메모리 구조를 그림으로 이해해야 알 수 있음) 
	-키워드 인수: 매개변수명을 기입하는 경우
